HTML — це мова розмітки, яка використовується для створення структури веб-сторінок. Кожна веб-сторінка починається з HTML.

HTML надає базові елементи: заголовки, параграфи, списки та посилання, які дозволяють організувати контент на сторінці.

Структура HTML є основою кожного сайту. Вона визначає, як різні частини сторінки будуть взаємодіяти одна з одною.
